Файл Readme программы для регистрации пользователей REGISTRECODE 2020 

25/02/2020 17:21:13

Этот документ содержит важные сведения, с которыми необходимо ознакомиться до установки программы REGISTRECODE 2020. 
В нем описывается, как получить последнюю версию электронной документации по REGISTRECODE 2020, как получить пакет обновления REGISTRECODE 2020, 
где найти помощь по установке, а также как получить дополнительные сведения о продукте в заметках о версии.


Выпущенное в январе 2020 г. приложение REGISTRECODE 2020 является основным продуктом компании RIELTINC, занимающейся предоставлением риелторских услуг.
Получить электронную документацию на приложение можно следующими способами:


             -Загрузить версию за январь 2020 со страницы загрузки электронной документации по REGISTRECODE. Дополнительные сведения см. на веб-узле RLTN в разделе 
             Загрузка и обновление электронной документации.


             -Просмотреть электронную документацию на веб-узле RLTN.

После установки требуется активировать лицензию

Активация лицензии REGISTRECODE 2020:

1. Устанавливаем программу и перезагружаем ПК ;
2. Отключаемся от сети;
3. Запускаем приложение и в появившемся окне жмем Enter serial number;
4. Соглашаемся с правилами;
5. Скачиваем и запускаем кряк- кейген с правами администратора;
6. Жмем кнопочку Patch;
7. Возвращаемся в REGISTRECODE 2020 и жмем кнопку Activate;
8. Откроется окно, в нем вводим любой из этих ключей - 666-69696969, 667-98989898, 400-45454545, 066-66666666, а в поле ключ продукта введите 128J1 и нажмите Next;
9. В следующем окне нажмите кнопку Back;
10. Отмечаем галочкой I have an activation code from Markthree;
11. Копируем полученный ключ запроса и вставляем его в кейген, жмем Generate;
12. Получаем код активации - его вставляем в окне программы, жмем Next и получаем полную лицензию.



               Использование программы REGISTRECODE 2020


1.При входе в программу необходимо зарегестрироваться в системе. Происходит это на отдельной странице "Регистрация".

2.После прохождения регистрации в соответсвии с требованиями данные о пользователи заносятся в систему.

3.При повторном входе в приложение требуется ввести лишь Логин и Пароль на странице Авторизация.

4.На пользовательской странице клиент может ознакомиться с  информацией об объектах недвижимости и ценами.

5.Также клиенту доступны различные фильтры для облегчения навигации, нужно лишь ввести в поля фильтров, что нужно найти.

6.Если вы являетесь администратором данной системы, то вы должны ввести соответствующие администратор логин и пароль.

7.После входа под "Администратором" открывается страница редактирования базы данных.

8.Администратор имеет право добавлять новые записи в БД (кнопка "Добавить") и удалять уже существующие (кнопка "Удалить").

9.Для сохранения изменений нужно нажать кнопку "Сохранить".
